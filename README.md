# 拓荒 / Homestead — 试玩版 Demo

白手起家的像素生存建造游戏：砍树、开垦、种田、做饭，在荒野中求生，一步步造出工具、电力与工业，甚至造火箭飞向其他星球。
A from-scratch pixel survival & building game — chop, till, farm, cook, industrialize, and launch a rocket to other worlds.

## ▶ 在线试玩 / Play in browser



> 单文件网页游戏，直接在浏览器里玩，无需下载。进度自动保存在浏览器本地。
> Runs entirely in the browser, no download. Progress is auto-saved locally.
> 试玩版进行到游戏内第 15 天结束。/ The demo ends on in-game Day 15.

## 🎮 操作 / Controls
- 移动 / Move：WASD 或方向键 (Arrow keys)
- 互动·采集 / Interact：空格 或 右键 (Space / Right-click)
- 背包 / Backpack：B　地图 / Map：M　菜单 / Menu：Esc
